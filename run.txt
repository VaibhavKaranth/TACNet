  Correct run order

  # 1. Install
  pip install -r requirements.txt

  # 2. Train everything (GPU recommended, ~hours on CPU)
  python main.py --mode run_all

  # OR quick run for CPU
  python main.py --mode run_all --quick

  # 3. Launch the demo
  python app_gradio.py

  The demo gracefully handles the case where models aren't trained yet — it shows a clear warning message with the    
  exact command to run.
